# Produktionskonfiguration SwissAirDry

## 1. Domain & URLs
```
Hauptdomain: vgnc.org
API Domain: api.vgnc.org
```

## 2. Server-Konfiguration

### Verzeichnisstruktur (absolut)
```
/sde3/
├── api/                    # API-Server
├── mqtt/                   # MQTT-Konfiguration
│   └── config/
│       └── mosquitto.conf
├── nextcloud/             # Nextcloud-Installation
│   └── apps/
│       └── swissairdry/
├── postgres/              # PostgreSQL-Daten
├── postgres_data/         # Persistente DB-Daten
├── docker-compose.yml
└── .env
```

### Docker Container & Ports

```yaml
services:
  api:
    ports:
      - "5000:5000"    # API Server
    networks:
      - swissairdry_network

  postgres:
    ports:
      - "5432:5432"    # PostgreSQL
    networks:
      - swissairdry_network

  mqtt:
    ports:
      - "1883:1883"    # MQTT Broker
      - "9001:9001"    # MQTT WebSocket
    networks:
      - swissairdry_network

  nextcloud:
    ports:
      - "8080:80"      # Nextcloud Web
    networks:
      - swissairdry_network
```

### Umgebungsvariablen (.env)
```bash
# API Konfiguration
API_PORT=5000
API_PREFIX=/api/v1
API_HOST=0.0.0.0
API_URL=https://api.vgnc.org

# Datenbank
POSTGRES_USER=swissairdry
POSTGRES_PASSWORD=swissairdry
POSTGRES_DB=swissairdry
POSTGRES_HOST=postgres
POSTGRES_PORT=5432

# MQTT
MQTT_BROKER=mqtt
MQTT_PORT=1883
MQTT_USERNAME=swissairdry
MQTT_PASSWORD=swissairdry
MQTT_TOPIC_PREFIX=swissairdry

# Nextcloud
NEXTCLOUD_URL=https://vgnc.org
NEXTCLOUD_USERNAME=admin
NEXTCLOUD_PASSWORD=admin
```
