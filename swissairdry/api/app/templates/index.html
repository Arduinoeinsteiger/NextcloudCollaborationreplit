Willkommen zur SwissAirDry API

Die SwissAirDry API ermöglicht die Steuerung und Überwachung von Trocknungsgeräten sowie die Verwaltung von Aufträgen und Gerätedaten. Diese API bildet das Herzstück des SwissAirDry-Ökosystems.

API-Dokumentation Gerätedaten System-Status ExApp Integration

Features

Gerätesteuerung

Fernsteuerung und Überwachung von SwissAirDry Trocknungsgeräten in Echtzeit.

Nextcloud ExApp Integration

Nahtlose Integration mit der Nextcloud ExApp für Projektmanagement und Auftragsabwicklung.

Sensordaten

Erfassung und Analyse von Umgebungsdaten wie Temperatur, Luftfeuchtigkeit und Energieverbrauch.

MQTT Unterstützung

Kommunikation mit Geräten über das MQTT-Protokoll für zuverlässige Datenübertragung.

API Bereiche

  • /api/devices - Verwaltung von Geräten
  • /api/device/{id}/data - Sensordaten abrufen und speichern
  • /api/device/{id}/command - Befehle an Geräte senden
  • /api/exapp - Nextcloud ExApp Integration
